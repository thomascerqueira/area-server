    URL: /auth/deleteUser

    Method: POST

    Description

     Use to delete a user in firebase with token

    Header

    {
  "tokenID": "'Bearer' + TokenID provided by firebase": string
}

    Success Response

    Code: 200 OK

    Content

    {
  "msg": "User deleted": string
}

    Error Response

    Status Condition Response
    401 Bad token 
    {
  "msg": "Bad format Token"
     or firebaseError
}
    500 If uid is wrong or error in firebase 
    firebase error
